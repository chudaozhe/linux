问题1:
150418 10:14:55 mysqld_safe mysqld from pid file /data/apps/mysql/data/localhost.localdomain.pid ended
解决办法:
查看3306端口是否被占用
或
由于sock(/data/apps/mysql/mysqld_3306.sock)在mysql目录，则nobody得有写的权限
chmod 753 /data/apps/mysql/
